# Sales Performance Analysis (Excel)

## Tools Used
- Microsoft Excel (Pivot Tables, Charts, Data Cleaning)

## Description
This project analyzes sales data to understand overall business performance. 
Data was cleaned and organized, then analyzed using pivot tables and charts.

## Key Insights
- Identified top-performing products
- Analyzed monthly sales trends
- Found regions with highest revenue
