# E-commerce Data Analysis (Excel + Power BI)

## Tools Used
- Excel (Data Cleaning)
- Power BI (Dashboard, Visualization)

## Description
This project focuses on analyzing e-commerce sales data. 
Excel was used for data cleaning, and Power BI was used to create an interactive dashboard.

## Key Insights
- Customer purchase behavior
- Best-selling categories
- Revenue trends over time

## Dashboard Features
- Sales overview
- Category-wise performance
- Monthly trends
