# Financial Data Analysis Dashboard (Power BI)

## Tools Used
- Power BI (Data Visualization, DAX)

## Description
This project analyzes financial data to track business performance and profitability. 
An interactive dashboard was created using Power BI.

## Key Insights
- Profit and loss analysis
- Expense tracking
- Revenue vs cost comparison

## Dashboard Features
- KPI metrics
- Financial trends
- Category-wise analysis
